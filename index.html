üå± Project Overview

This project detects soil moisture percentage using a soil moisture sensor and an Arduino. Results are shown on an OLED display for quick monitoring and for building smart irrigation systems.

The Soil Moisture Detection System is a smart, Arduino-based project designed to measure and display the moisture content (percentage) of soil in real time. It helps farmers, gardeners, and researchers monitor soil water levels effectively and ensure that plants receive the right amount of water ‚Äî neither too dry nor overwatered.

The system uses a Soil Moisture Sensor to detect the amount of water present in the soil. The sensor works on the principle of electrical conductivity: wet soil conducts more electricity (lower resistance), while dry soil conducts less electricity (higher resistance). The sensor produces an analog voltage that reflects this conductivity.

The analog output from the sensor is read by the Arduino Uno, which converts the raw reading into a calibrated percentage value (0‚Äì100%) using a mapping/calibration formula. That calculated moisture percentage is then displayed on a 0.96-inch OLED display for easy visualization and quick status checks.

This project is an excellent example of integrating IoT and embedded systems for smart agriculture ‚Äî it can be expanded to automatically control irrigation systems based on real-time moisture data, log measurements to a server, or trigger alerts when levels fall outside target ranges.

Arduino Uno

Main microcontroller board

  • Microcontroller: ATmega328P
  • Operating Voltage: 5V (USB) ‚Äî Vin accepts up to 12V
  • Digital I/O: 14 (6 PWM); Analog inputs: 6 (A0‚ÄìA5)
  • Typical use: read analog sensor on A0, drive OLED via I2C (A4/A5)
  • Tip: Use common GND and avoid powering motors from the 5V pin directly.

Soil Moisture Sensor

Detects water content in soil

  • Type: resistive probe (or capacitive module) ‚Äî this example shows the common resistive probe.
  • Operating Voltage: 3.3‚Äì5V; Output: Analog voltage proportional to moisture.
  • Connect: VCC ‚Üí 5V, GND ‚Üí GND, AOUT ‚Üí A0.
  • Calibration: dry‚âà1023, wet‚âà~200 (adjust map() limits in code to match your sensor).
  • Tip: For long-term use, prefer capacitive sensors to avoid corrosion.

Breadboard

Connects components without soldering

  • Rows/columns: use the central rows for components and side rails for power (red/blue).
  • Tip: Keep power rails consistent (5V and GND) and avoid long jumper runs for reliability.
  • Use bus strips for distributing 5V and GND to many modules.

Jumper Wires

  • Types: male‚Äìmale, male‚Äìfemale, female‚Äìfemale. Pick the correct type for your headers.
  • Quality: thicker wires and solid connectors reduce accidental disconnections.

OLED Display (SSD1306)

üíª Arduino Code

#include <Wire.h>
#include <Adafruit_GFX.h>
#include <Adafruit_SSD1306.h>

#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, -1);

const int sensorPin = A0;
int sensorValue = 0;
int moisturePercent = 0;

void setup() {
  Serial.begin(9600);
  if(!display.begin(SSD1306_SWITCHCAPVCC, 0x3C)) {
    Serial.println(F("OLED failed"));
    for(;;);
  }
  display.clearDisplay();
  display.setTextSize(1);
  display.setTextColor(WHITE);
}

void loop() {
  sensorValue = analogRead(sensorPin);
  moisturePercent = map(sensorValue, 1023, 200, 0, 100);
  moisturePercent = constrain(moisturePercent, 0, 100);
  
  Serial.print("Soil Moisture: ");
  Serial.print(moisturePercent);
  Serial.println("%");

  display.clearDisplay();
  display.setCursor(0, 20);
  display.setTextSize(2);
  display.print("Moisture:");
  display.setCursor(0, 45);
  display.print(moisturePercent);
  display.print("%");
  display.display();

  delay(1000);
}
